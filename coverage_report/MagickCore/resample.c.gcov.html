     808                 :             :    *

     812                 :             :    * under consideration is defined as follows:
     813                 :             :    *

     824                 :             :    * input changes in the horizontal direction, and the vector [B,D]
     825                 :             :    * is the tangent vector corresponding to input changes in the
     826                 :             :    * vertical direction.
     827                 :             :    *
